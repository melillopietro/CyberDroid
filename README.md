# CyberDroid

Welcome to CyberDroid VM - a fully customizable, Linux-based security distribution for Android malware analysis.

---
## Tools

### Static Malware Analysis :
* Androbugs
* MARA Framework
* MobSF`
### Dynamic Malware Analysis : 
* wireshark
* Burpsuite
* ADB
### Decompiler: 
* APKtool 
* Dex2jar
### Crypto:
* Stegide
* Hashcat
* Cyberche
